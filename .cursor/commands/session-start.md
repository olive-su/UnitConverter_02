# /session-start

Load context to begin or resume a session.

## Steps
1. Read `.cursor/rules/00-ground-rules.mdc`.
2. Read `harness/sessions/INDEX.md`.
3. Read the most recent session log in the index.
4. Read `README.md` and `Ref/02-Goal.md` for project context.
5. Restate the current goal and open items.
6. Propose a short, ordered task list before acting.
