## 2. R-G-I-O 워크북

| | 내용 |
|---|---|
| **R — Role** | 길이 `unit:value`를 **여러 단위로 동시에** 맞춰야 하는 실습생·초급 개발자 |
| **G — Goal** | **한 입력**으로 **전 등록 단위** 변환값을 **meter SSOT 비율**로 일관 출력; 잘못된 입력은 **명확한 오류** |
| **I — Input** | 문자열 `unit:value` (예: `meter:2.5`) |
| **O — Output** | 전 단위 변환 결과(기본 table) 또는 형식 오류·음수·미지 단위 **오류 메시지** |

### 주제 (1문장)

> **한 `unit:value` 입력으로 meter·feet·yard를 meter 기준 비율로 일관 변환하고, 잘못된 입력은 거부하는 CLI와 테스트 루프를 만든다.**
