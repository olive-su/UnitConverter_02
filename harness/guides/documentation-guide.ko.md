# 문서 작성 가이드 (Documentation Guide)

English version: [documentation-guide.md](documentation-guide.md).

## 개요
- 이 프로젝트의 문서 작성 방법.
- 목표: 간결·일관·이중언어로 빠르게 훑어볼 수 있는 문서.

## 이중언어 규칙
- 모든 문서를 `name.md`(영문)와 `name.ko.md`(한국어)로 작성.
- 같은 변경에서 두 버전 갱신, 동기화 유지.
- 상단에서 다른 언어 버전으로 링크.

## 스타일
- 개조식·간결. 긴 문장보다 구문 단위.
- 한 페이지 한 목표. 짧은 개요로 시작.
- 이모지 금지.

## 구조 체크리스트
- 한 페이지 한 목표. H4를 넘으면 분리.
- 핵심 결론을 담은 개요로 시작.
- 예측 가능한 일관된 제목과 용어.
- 가치 우선, 부수 정보는 후순위.
- 제목: 간결·키워드 포함·평서문.

## 문장 체크리스트
- 필요한 정보만. 한 줄에 한 아이디어.
- 메타 담화 제거 (예: "앞서 설명했듯이").
- 구체적으로. 행동 동사 사용.
- 일관되게. 약어는 첫 등장 시 풀어 쓰기.
- 주어를 분명히. 능동형 선호.
