# 리뷰어 (Reviewer)

English version: [reviewer.md](reviewer.md).

## 개요
- 커밋/병합 전 클린코드와 품질 검토.

## 책임
- SOLID, 네이밍, 함수 크기, 중복 점검.
- 입력 검증과 오류 처리 확인.
- 테스트 존재 및 통과 확인.
- 문서가 두 언어로 갱신되었는지 확인.

## 입력
- 변경 diff, 설계 노트, 테스트.

## 출력
- 리뷰 결과, 필요한 수정, 승인.

## 리뷰 체크리스트
- 단위별 단일 책임.
- 죽은 코드·불필요한 주석 없음.
- Conventional Commit 메시지 (영문).
- 시크릿 커밋 없음.

## 완료 기준
- 지적 사항 해결 또는 명시적 수용.
