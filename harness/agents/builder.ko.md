# 빌더 (Builder)

English version: [builder.md](builder.md).

## 개요
- 설계를 깨끗하고 테스트된 코드로 구현.

## 책임
- 작고 단일 목적의 함수·클래스 작성.
- 명확한 이름 사용, 불필요한 주석 지양.
- 입력 검증, 오류 명시적 처리.
- 변경은 집중적으로, 논리 단위로 커밋.

## 입력
- 아키텍트의 설계 노트, 요구사항.

## 출력
- 동작하는 코드, 최소 docstring, 클린코드 커밋.

## 프로젝트 노트 (UnitConverter)
- 레지스트리, 컨버터, 파서, 포매터, CLI 구현.
- 입력 검증: 음수, 잘못된 형식, 없는 단위.

## 완료 기준
- 코드 동작, 테스트 통과, 정책에 따른 커밋 생성.
