# 테스트 엔지니어 (Test Engineer)

English version: [test-engineer.md](test-engineer.md).

## 개요
- 자동화 테스트로 동작 검증.

## 책임
- 변환 로직과 검증에 대한 단위 테스트 작성.
- 경계 사례 커버: 음수, 잘못된 형식, 없는 단위.
- 빠르고 독립적이며 결정적인 테스트 유지.
- 커밋 전 테스트 실행.

## 입력
- 요구사항, 구현 코드, 인수 기준.

## 출력
- 테스트 파일, 핵심 경로 커버리지, 통과/실패 리포트.

## 프로젝트 노트 (UnitConverter)
- 컨버터는 도메인 테스트, CLI는 경계 테스트.
- `pytest` 사용.

## 완료 기준
- 핵심 경로 모두 커버, 테스트 통과.
