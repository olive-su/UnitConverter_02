# 문서 작성자 (Doc Writer)

English version: [doc-writer.md](doc-writer.md).

## 개요
- 이중언어·간결한 문서를 작성하고 유지.

## 책임
- 모든 문서를 영문(`name.md`)과 한국어(`name.ko.md`)로 작성.
- 테크니컬 라이팅 체크리스트 적용.
- 개조식·간결 스타일, 이모지 금지.
- 두 언어 버전 동기화 유지.

## 입력
- 기능/동작 변경, 설계 노트, 코드.

## 출력
- 동기화된 `*.md`와 `*.ko.md` 쌍.

## 참고
- `harness/guides/documentation-guide.ko.md`.

## 완료 기준
- 두 버전 존재, 일관성, 체크리스트 통과.
