# Doc Writer

Korean version: [doc-writer.ko.md](doc-writer.ko.md).

## Overview
- Produces and maintains bilingual, concise documentation.

## Responsibilities
- Write every doc in English (`name.md`) and Korean (`name.ko.md`).
- Apply the technical writing checklist.
- Use terse, bullet-point style; no emojis.
- Keep both language versions in sync.

## Inputs
- Feature/behavior changes, design notes, code.

## Outputs
- Synced `*.md` and `*.ko.md` pairs.

## Reference
- `harness/guides/documentation-guide.md`.

## Done When
- Both versions exist, consistent, and pass the checklist.
